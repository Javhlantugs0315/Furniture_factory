# Система управления мебельной фабрикой
# Описание проекта
Данный проект выполнен в рамках учебной дисциплины «Интернет-программирование». Он реализует информационную систему для мебельной фабрики (вариант 13).

Система позволяет:
- вести реестр цехов и начальников;
- хранить информацию о рабочих в каждом цехе;
- управлять заказами на корпусную и мягкую мебель;
- отслеживать этапы выполнения заказов в разных цехах;
- централизованно управлять данными через административную панель Django.

Проект предназначен для администратора фабрики.
# Установка и запуск

1. Клонировать репозиторий:

git clone <https://github.com/Javhlantugs0315/Furniture_factory.git>
cd furniture_factory

2. Установить зависимости (рекомендуется виртуальное окружение):

pip install django
pip install python-dateutil

3. Выполнить миграции:

python manage.py makemigrations
python manage.py migrate

4. Создать суперпользователя для доступа в админку:

python manage.py createsuperuser

Для демонстрации работы системы можно использовать тестовую учетную запись администратора:

Логин: admin

Пароль: admin12345

5. Запустить сервер разработки:

python manage.py runserver

6. Перейти в браузере:

http://127.0.0.1:8000/
